## Para ver el proyecto en VSC

🔹 En Visual Studio Code, en la ventana del terminal:
```bash
npm install
```
Para levantar el proyecto:

```bash
npm run start
```

Abra [http://localhost:3000](http://localhost:3000) en su browser (si no se abrio de forma automatica).
